# simpleRSA

## Использование

#### Linux:
bash
```
./LabRSA {путь_к_входному_файлу} {путь_к_выходному_файлу} {простое_число_p} {простое_число_q}
```

#### Windows:
bat(shell)
```
LabRSA.exe {путь_к_входному_файлу} {путь_к_выходному_файлу} {простое_число_p} {простое_число_q}
```
